DirectX 11을 이용해 보는 것이 목표인 프로젝트입니다.

주요 특징은 다음과 같습니다.
- 디버드 렌더링
- Effect 프레임워크를 사용X, HLSL 리플렉션을 직접 적용.
- FBX SDK를 이용하여 3D 모델링 로드, 애니메이션 재생
- NVIDIA PhysX를 이용한 물리 시뮬레이션
